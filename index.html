My Projects Some Of My Works

Pentester Web Application

Project implementing our security and web programming skills (html, css, php) in order to implement a web application allowing testing of different types of vulnerabilities (XSS, SQL, CSP, CORS, CSRF) on a site given by the user. The final project included thousands of injections which were tested quickly thanks to the use of threads. Finally, the site included documentation pages on each type of vulnerability as well as a chatGPT chatbox.

Re-implementation of a game (CAESER III) and implementation of a multiplayer mode

This project, separated into two parts, aimed to recreate the famous CAESER III using python and the Pygame library. From the logic to the menus, everything was recoded and we added game mechanics. Finally, we created a co-op style multiplayer mode using two processes, one in Python and the other in C, communicating packets using TCP protocols.

Optimizing Data Anonymization and Re-Identification

This project was a competition among INSA schools, implemented in Python, and unfolded in three parts. First, the anonymization of a database containing over 30,000,000 records across 4 columns (Id, Date, Longitude, Latitude) following specific metrics. Then, the re-identification of other competitors' databases, and finally, the development of our own metrics. In essence, this project forced me to optimize my code (using multiprocessing, threads, etc.) to handle a large volume of data with hardware that was not necessarily suited for the task. Moreover, I discovered new Python libraries such as pandas for database management. It was also interesting to adopt both the defender's and attacker's perspectives to gain a better understanding of the operations.
